*Launcher Tab*

Many users require the ability to quickly login with different accounts, or on different shards.
This tab allows the user to create and save profiles with appropriate login information.
The user can then select the profile of his/her choice and quickly login with the client of his/her choice.

Account List
The box to the left of the tab contains the list of saved accounts. Select one of these to lauch your client with the appropriate information.

Launch 2D
Launches the default 2D client with the account information selected from the account list.

Launch 3D
Launches the default 3D client with the account information selected from the account list.
*Note* 3D client not supported in more recent UO installation.

Launch Alt
Launches the profile specific alternate client with the account information selected from the account list.

Auto Patcher
Launches the Ultima Online's auto patcher program.

New
Allows to create a new account profile.

Edit
Allows editing of the selected account profile from the account list.

Delete
Deletes the selected account profile from the account list.

Account Information
Name - Enter the desired name for the account profile.
Account - This is your UO account name.
Password - This is your password for this UO account.
IP - This is the IP to the shard of this account.
Port - This is the port used by the shard of this account.

Save
Save the new or edited account profile.

Cancel
Cancels the creation of the new profile or changes to edited account profile.

Alternate Client
This is the path to an alternate client you would like to use with this account.
This is set to the default 2D client if not changed.

Browse
Allows you to browse your files to pick the alternate client of your choice.
