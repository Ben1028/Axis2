*Travel Tab*

This tab is designed to make traveling easyer. Categories of locations can be found in the left window. Single click on a category will make the locations appear in the center window. Single clicking a location will recenter the map view, while double clicking a location will activate the Go command.

Go
This will send you to the selected location. Co-ordinates of the location are found at the bottom-center of the tab.

Send
This will allow you to select any player/NPC to be sent to the selected location.

Add
This will bring up a small dialog to allow you to add custom locations to the list. Details can be found here...
*Note* These are saved in the registry and loaded with every profile.

Edit
This will bring up a small dialog to allow you to edit custom location. Details can be found here...
*Note* Only possible to edit locations created with the Add command.

Del
This allows you to delete custom locations.
*Note* Only possible to delete locations created with the Add command.

 

Where
This will send the .where command in game to show you your location.

Zoom in
This will zoom in the map view each time it's pressed for up to 4x bigger then regular scale.

Zoom out
This will zoom out the map view each time it's pressed for up to 4x smaller then regular scale.

Pos
This will recenter the map view with your current location in game.
*Note* Client must be running.

Track
This will recenter the map view with your current location in game every 1 second.
*Note* Client must be running.

Map
Changing this variable will change the map displayed in the map view. This is also changed when a location is selected.

Search
This will bring up a small dialog that allows to perfom a search for loaded locations. Details can be found here...

Map view
When "Show map display" is activated, this will show a graphique view of the UO world. When single left-clicked, the map is recentered with the white cross-hair to the location selected. The map can also be draged around while holding down the right mouse button. Scroll bars are provided to scroll the map up/down, left/right. If "Show spawn points" is activated, every spawn points will be shown on the map with a cross-hair in either red for npcs or yellow for items by default.

World map
This will bring up a small dialog that displays the entire selected map. A single left click on this map will recenter the map view to the selected location.
