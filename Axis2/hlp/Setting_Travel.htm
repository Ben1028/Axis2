*Settings - Travel*

All settings to modify Axis2's Travel Tab can be changed here. Changes are automatically saved and restored on startup.

Show map display
This will turn on/off the map display on the Travel Tab. (On by default).

Draw statics
This will display static items on the map display. (On by default).

Use map and static dif files
This will apply the map and static *.dif files on the map display. (Off by default).
*Note* This option is not needed for clients 6.0.0 and newer.

Show spawn points
This will show the location of the spawn points on the map display. A worldsave must be loaded for this to work. (On by default).

NPC spawn color
This is the color used for NPC spawn point display. (Red by default).

Item spawn color
This is the color used for item spawn point display. (Yellow by default).
