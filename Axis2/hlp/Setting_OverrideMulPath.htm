*Settings - Override Mul Paths*

All settings to override mul paths. Changes are automatically saved and restored on startup.

File Paths
This is the list of files that Axis2 needs to load. (.mul , .idx , .def).
*Note* You must select a file to modify it or reset it to it’s default path.

Browse
This allows you to browse files to modify the selected path.

Reset Default Path
This will reset the selected path to it’s default path.
