*Settings - File Paths*

All settings to modify the default client and mul paths. Changes are automatically saved and restored on startup.

Default ultima online client
This is the path to the default Ultima Online client. The path is set automatically from the registry if not altered.
•Browse - This allows you to browse for a different client path.

Default mulfile path
This is the path to the default Ultima Online mulfiles. By default, it is set to the default client path.
•Browse - This allows you to browse for a different mulfile path.
•Use same path as default client - This will set the mulfile path to the same as the client path.
