*Commands Tab*

This tab allow the user to define and store commonly used command. With the ability to create 36 buttons per tab, and create as many tabs as you which, you can store an infinit amount of commands.

Button Control
•Caption - Enter a descriptive name for the button.
•Command - This is where you type the command to be stored. This can be anything, from regular speech to commands.
*Remember* Commands must start with a period "."
•Add - This button will create your new command on the current tab.
•Edit - This will ask you to select a button to edit. when selected, it will replace the buttons information with the one specified in the caption and command field.
•Remove - This will ask you to select a button to be removed. A confirmation is required to prevent accidental removal of buttons.

Tab Control
•Tab Label - Enter a descriptive name for the tab.
•Add Tab - Will add a new tab with the specified label.
•Del Tab - Will delete the currently selected tab. A confirmation is required to prevent accidental removal of tabs.
*Note* The last tab can not be deleted!
•Clear Tab - Will clear all buttons from the currently selected tab. A confirmation is required to prevent accidental clearing of tabs.
