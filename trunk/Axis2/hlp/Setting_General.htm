*Settings - General*

All settings to modify Axis2's general behavior can be changed here. Changes are automatically saved and restored on startup.

Basic Axis Settings
•Allow multiple instances of Axis - When turned on, will allow to run multiple instances of Axis at the same time. (Off by default).
•Always on top - This allows to keep Axis the top most window on your desktop. (Off by default).
•Allow close from system menu - This allows you to close Axis with the red box with the X on the top right corner. (On by default).
*Note* If off, you must close it with the Close Axis menu option on the right-most side.
•Load default profile on startup - When on, Axis will automatically load the default profile you create on startup. (Off by default).
•Start with ??? tab - This tells Axis which tab to set in focus on startup. (General tab by default)

Reset settings
This will return all settings to their default state, including Tab settings.

*Note* Override Paths are not reseted.
