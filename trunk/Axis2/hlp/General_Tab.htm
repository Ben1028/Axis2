*General Tab*

This tab contains general purpose commands. Things like GM Pages, Discipline, World teaks, and even console access to name a few.

Information
•Admin - Will send the .admin command to access the in-game admin menu.
•Info - Will send the .info command to get in-game info on items or pc/npcs.
•Edit - Will send the .edit command to access all layers on a pc/npc.
•Clients - This will show you the number of currently logged-in players.
•Serv Info - This will show you miscellanious information about the server.
•Version - This will show you the server version.

Misc
•Link - This command allows you to link to items together. It is often used on doors to make them open at the same time.
•Flip - This will switch the dispid of an item to the next one in it's dupelist. Most often, it is just a new direction to the item.
•Shrink - This will shrink a pet for easy transportation. GMs can shrink any nps, not just pets.
•Dupe - This will duplicate the target item.
•Remove - This will remove the target item or npc. Players can not be removed this way.
•Nuke - This allows you to clear an area between 2 selected points on the map.

Vendors
•Buy - This will open the buy menu of a vendor.
•Sell - This will open the sell menu of a vendor.
•Inventory - This will open the targeted vendor's inventory box. These are items the vendor will sell.
•Purchases - This will open the targeted vendor's purchase box. This is where the items he/she buys are located.
•Samples - This will open the targeted vendor's sample box. These are items the vendor will buy.
•Restock - This will restock the targeted vendor.

Weather
•Snow - This will make it snow in the current sector.
•Rain - This will make it rain in the current sector.
•Dry - This will turn off weather effects in the current sector.
•Light Level - This will set the light level in the current sector to the level selected with the slider control below.

Self
•Invul - This will turn on/off invulnerability on yourself.
•Allmove - This will turn on/off the ability to move everything
•Invisible - This will turn on/off invisibility on yourself.
•Fix - This will adjust your z co-ordinate to the highest surface below yourself.
•Tele - This allows you to select a location to teleport to.
•Hear All - This will turn on/off the ability to hear everything that is said on the server.
•GM Toggle - This will turn on/off GM mode.
•Detail - This will turn on/off combat detail messages on yourself.
•NightSight - This will turn on/off the ability to allways see with daylight.
•Debug - This will turn on/off debug mode.

Discipline
•Jail - This allows you to select a player to send to jail.
•Forgive - This allows you to select a player to release from jail.
•Kill - This simply kills the selected target.
•Disconnect - This will disconnect from the server the selected player.
•Resurect - This will resurect the selected player.

GM Page
•On - This allows you to hear GM pages.
•Off - This will turn off the ability to hear GM pages.
•List - This lists all pending pages.
•Queue - This put the current page back into pending queue.
•Go Player - This will send you to the location of the player that sent the page.
•Go Origin - This will send you to the location the page was created from.
•Disconnect - This will disconnect the player that sent the page.
•Jail - This will jail the player that sent the page.
•Ban - This will disconnect and ban the player that sent the page.
•Delete - This will delete the curent page.

Save World
This command will start a world save.

*Note* No static items (items with attr_static) are saved.

Save Statics
This will start a static save.

*Note* Only static items are saved.

Resync
This will resync the server to allow changes to scripts to take place.

Restock all NPC
This will restock all NPC.
