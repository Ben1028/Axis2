*Spawn Tab*

This tab is designed to facilitate adding NPCs and make spawning an effortless task. Categories of NPCs can be found in the left window. Single click on a category will make the list of NPCs appear in the center window. Single clicking an NPC will display it in the rightmost window if the Show NPCs Display setting is turned on.

Summon
This allows you to summon the selected NPC.

Remove
This allows you to select an in-game NPC to remove.
*Note* Players can not be removed this way

Shrink
This allows you to select an in-game NPC to shrink. It creates a small statuette of the targeted NPC.

Freeze
This allows you to select an in-game NPC to freeze.

Set Home
This allows you to select an in-game NPC to set their home position. The co-ordinates used are the ones where the NPC is standing at the moment of targeting.

Set Homedist
This allows you to select an in-game NPC to set their home traveling distance. The distance used is the one provided in the text box next to this button.

Search
This will bring up a small dialog that allows to perfom a search for loaded NPCs. Details can be found here...

NPC Spawn
Amount - The maximum number of creatures to be spawned from this point. The creatures are spawned one at a time, up to this value. If a creature spawned from this point dies, it will be respawned somewhere between the mint time and max time.
Min Time - Minimum time between spawns.
Max Time - Maximum time between spawns.
Max Dist - The maximum distance for the creature to be spawned from this point. This will also be the homedist value for the creature. If this value is 0, the creature will be spawned at this location, but will be able to wander anywhere it chooses. The spawns will not wander farther from its spawn point than its Max Distance.
Place - Places the spawn point.The spawn point will appear as a worldgem.
*Note* The spawn point is not active at this time.
Init - Initializes the spawnpoint using the parameters specified.
*Note* You do not need to target the worldgem as AXIS will do this automatically.
